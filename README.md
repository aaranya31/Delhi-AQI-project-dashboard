# Delhi AQI Dashboard

## Overview

This project presents an interactive Power BI dashboard for analyzing Delhi Air Quality Index (AQI) data. The dashboard enables users to explore pollution trends, identify high-risk periods, and gain insights into air quality using dynamic visualizations and filters.

---

## Tools Used

- Power BI
- SQL
- Microsoft Excel

---

## Dashboard Features

- KPI Cards
- AQI Trend Analysis
- Monthly Comparison
- Category-wise AQI Distribution
- Dynamic Filters
- Conditional Formatting
- Interactive Visualizations

---

## Dataset

The dataset contains Delhi Air Quality Index observations used for analysis and dashboard creation.

---

## Key Insights

- Identified monthly AQI trends.
- Compared pollution levels across AQI categories.
- Highlighted periods with unsafe air quality.
- Built an interactive dashboard for efficient data exploration.
